# Notebooks

Use this folder for exploratory notebooks only.

Recommended notebook order:

| Notebook | Purpose |
|---|---|
| `01_f8_uc4_visual_checks.ipynb` | Validate cleaned data visually |
| `02_f9_problem_definition.ipynb` | Explore target definitions before ML |
| `03_f9_baseline_models.ipynb` | Train simple baseline models after the data pipeline is validated |

Keep production logic in `src/` and `scripts/`, not only in notebooks.
